## Exercise 7.06

For each of the following items of data, specify which one of the types `char`, `short`, `int`, or `long` is the smallest one guaranteed to be large enough to store the item.

```
(a) Days in a month
(b) Days in a year
(c) Minutes in a day
(d) Seconds in a day
```

## Solution

(a) 31, `char`  
(b) 365, `short`  
(c) 1440, `short`  
(d) 86400, `long`
